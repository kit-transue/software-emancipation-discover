New features in MKS Engineer Integrity 7.3 include:

 

Feature
Provides

C/C++/Java parser improvements

·        30% improvement in build time on small models and up to 60% on larger models

 

 

 

Impact analysis

·        Additional tools for understanding the impact of change to software

·        Method for developers to review the points of concern related to a given change

·        New reporting capabilities

·        Documentation explaining the different ways the impact of a change is tracked

 

 

 

Change Phase I, GUI, wizards, etc.

·        Graphical assistant for creating Tree Pattern Matching expressions

·        New capabilities for the manual configuration of small to midsize projects
Basic SIS support

Hardware and Software Requirements

 

Source Analysis works on the following platforms:

·        Windows NT

·        Windows 2000

This version of Source Analysis has the following minimum system requirements:

·        Pentium 200 with 128 megs of memory

·        Microsoft Visual C++, version 5.x, 6.x, or 7.x

Editor support:

·        Microsoft Visual C++

·        Emacs

·        VIM

·        JBuilder 3.5, 4.0, 5.0
