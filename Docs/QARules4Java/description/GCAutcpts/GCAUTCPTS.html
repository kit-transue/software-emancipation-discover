GCAUTCPTS - GC Avoid Unnecessary Temporaries when Convert Primitive Types to String.

Java provides wrapping classes for primitive types. Those classes provide a static method toString (...) to convert primitive types into their String equivalent. Use this method instead of creating an object of the wrapping class and then using the instance toString () method.

Wrong

public class AUTCPTS {
  String operFoo (int var) {
    return new Integer (var).toString ();
  }
}

Tip: Use a static method toString (...) to convert primitive types into their String equivalent.

Right

public class AUTCPTS {
  String operFoo (int var) {
    return Integer.toString (var);
  }
}
