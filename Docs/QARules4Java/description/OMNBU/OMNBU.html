OMNBU - Operator '?:' May Not Be Used

The operator '?:' makes the code harder to read, than the alternative form with an if-statement.

Wrong

void func (int a) {
    int b = (a == 10) ? 20 : 30;
}

Tip: Replace '?:' operator with the appropriate if-else statement.

Right

void func (int a) {
    if (a == 10)
        b = 20;
    else
        b = 30;
}
