UAAO - Use Abbreviated Assignment Operator

Use the abbreviated assignment operator in order to write programs more rapidly. Also some compilers run faster when you do so.

Wrong

void oper () {
    int i = 0;
    i = i + 20;
    i = 30 * i;
}

Right

void oper () {
    int i = 0;
    i += 20;
    i *= 30;
}
