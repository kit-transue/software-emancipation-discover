UC - Unnecessary Casts

Checks for the use of type casts that are not necessary.

Wrong

class Animal {}
class Monkey extends Animal {
    void func () {
        int i;
        float f = (float) i;

        Monkey e1;
        Monkey e2 = (Monkey) e1;

        Animal a;
        Monkey e;
        a = (Animal) e;
    }
}

Tip: Delete unnecessary cast to improve readability.
