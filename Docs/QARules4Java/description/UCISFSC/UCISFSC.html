UCISFSC - Use char instead of String for one character string concatenation.

Instead of concatenating one-character String constants, replace them with character constants for optimization.

Wrong

public class UCISFSC {
  public void oper(String s) {
    String string = s + "d";
  }
}

Tip: Replace them with character constants for optimization.

Right

public class UCISFSC {
  public void oper(String s) {
    String string = s + 'd';
  }
}
