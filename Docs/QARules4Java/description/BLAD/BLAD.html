BLAD - Badly Located Array Declarators

Array declarators must be placed next to the type descriptor of their component type.

Wrong

class BLAD {
    int attr[];
    int oper (int param[]) [] {
        int var[][];
        // do something
    }
}

Tip:

Right

class BLAD {
    int[] attr;
    int[] oper (int[] param) {
        int[][] var;
        // do something
    }
}
