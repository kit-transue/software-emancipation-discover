GCFMUL - GC - finalize() method should not unregister listeners

The finalize() method gets only called when there are no more references to the object. If the listeners are removed in the finalize() method the object will not be removed during garbage collection.

Wrong

public class GCFMUL extends Panel {
  public void finalize() {
    remove (_button);
    super.finalize();
  }
  private Button _button = new Button();
}

Tip: Do not call methods that remove listeners in the finalize() method body.
