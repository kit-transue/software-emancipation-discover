UTETACM - Use 'this' Explicitly To Access Class Members

Tries to make you use 'this' explicitly to access class members. Using same class members' and parameters' names makes references confusing.

Wrong

class UTETACM {
    int attr = 10;
    void func () {
        // do something
    }
    void oper () {
        func();
        attr = 20;
    }
}

Right

class UTETACM {
    int attr = 10;
    void func () {
        // do something
    }
    void oper () {
        this.func();
        this.attr = 20;
    }
}
