PCOLSOE - Place Constants On the Left Side Of Expression.

Place constants on the left side of comparisons/expression. By placing constants on the left side of comparisons the compiler will catch it if you accidentally use assignment instead of comparison.

Wrong

public class PCOLSOE {
  public void testOper (int something) {
    if (something == 5) {}
  }
}

Tip: Place constants on the left side of comparisons.

Right

public class PCOLSOE {
  public void testMethod (int something) {
    if (5 == something ) {}
  }
}
