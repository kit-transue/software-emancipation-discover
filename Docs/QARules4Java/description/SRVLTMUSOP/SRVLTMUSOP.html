Servlet API - Minimize use of System.out.println() or System.err.println().

System.out.println statements and similar constructs synchronize processing for the duration of disk I/O, they can significantly slow throughput.

Wrong

package com.swemancipation.servlet;

import javax.servlet.*;
import javax.servlet.http.*;

public class DemoDebugPrintServlet extends HttpServlet {
  public void service () {
    System.out.println ("starting service");
  }
}

Tip: Implement option to switch off printing debug information in production environment.

Right

package com.swemancipation.servlet;

import javax.servlet.*;
import javax.servlet.http.*;

public class BetterDemoDebugPrintServlet extends HttpServlet {
  private final static boolean DEBUG_ON = false;
  public void service () {
    // activate tracing only when absolutely needed.
    if (DEBUG_ON) {
      System.out.println ("starting service");
    }
  }
}
