ITLRO - Iterate Through Loops in Reverse Order.

Avoid using complex expressions as repeat conditions within loops.

Wrong

void oper () {
    for (int i = 0; i < list.length(); i++) {
        list.get(i).process();
    }
}

Tip: You may have access to elements of collection in reverse order, if this way don't break logic of your application.

Right

void oper () {
    for (int i = list.length(); i >= 0; i--) {
        list.get(i).process();
    }
}

See also CLE/Complex Loop Expressions.
