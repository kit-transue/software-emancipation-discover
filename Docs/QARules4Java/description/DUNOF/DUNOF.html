DUNOF - Don't Use Negation Operator Frequently

The negation operator(!) slows down the readability of the program, so it is recommended that you do not use it frequently. Caliper will report an warning for methods with 3 or more negation operators.

Wrong

public class DUNOF {
  boolean oper (boolean a, boolean b) {
    if (!a)
      return !a;
    else
     return !b;
  }
}

Tip: Do not use the negation operator if possible.
