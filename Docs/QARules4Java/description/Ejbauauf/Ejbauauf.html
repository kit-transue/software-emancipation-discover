Always use accessors to update JavaBeans properties.

When accessing a JavaBeans property, even internal access must be through an accessor. This is most important when changing the value.

Wrong

package com.swemancipation.ejbtest;

import javax.ejb.*;

public class AUAUF implements SessionBean{
  public int method (boolean b, int size) {
    if (b) { _size = size; }
    return getSize ();
  }
  public int getSize () { return _size; }
  public void setSize (int size) { _size = size; }
  public void setSessionContext (SessionContext ctx) {}
  public void ejbCreate () {}
  public void ejbActivate () {}
  public void ejbPassivate () {}
  public void ejbRemove () {}
  public int _size;
}

Tip: Do not access JavaBeans field directly. Always use accessors.

Right

package com.swemancipation.ejbtest;

import javax.ejb.*;

public class AUAUF implements SessionBean{
  public int method (boolean b, int size) {
    if (b) { setSize( size ); }
    return getSize ();
  }
  public int getSize () { return _size; }
  public void setSize (int size) { _size = size; }
  public void setSessionContext (SessionContext ctx) {}
  public void ejbCreate () {}
  public void ejbActivate () {}
  public void ejbPassivate () {}
  public void ejbRemove () {}
  public int _size;
}
