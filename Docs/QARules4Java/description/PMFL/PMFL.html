PMFL - Put the Main Function Last

Tries to make the program comply with various coding standards regarding the form of class definitions.

Wrong

public class PMFL {
    void func1 () {}
    public static void main (String args[]) {}
    void func2 () {}
}

Right

public class PMFL {
    void func1 () {}
    void func2 () {}
    public static void main (String args[]) {}
}
