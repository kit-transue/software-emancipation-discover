AHIA - Avoid Hiding Inherited Attributes

Detects when attributes declared in child classes hide inherited attributes.

Wrong

class Monkey extends Animal {
    int attr1;
    // something...;
}
class Animal {
    int attr1;
}

Tip: Rename child class attribute.

Right

class Monkey extends Animal {
    int mnkyAttr1;
    // something...;
}
class Animal {
    int attr1;
}
