ULV - 'Use Local Variables whenever possible.

When variables are accessed frequently, consider where these variables are accessed from. Is the variable static, local, or an field of class variable? static and field of class variables takes about two to three times longer to access than local variables.

Wrong

public class ULV {
  void oper1 (int[] values) {
    for (int i=0; i < value.length; i++) {
      _sum += value[i];
    }
  }
  void oper2 (int[] values) {
    for (int i=0; i < value.length; i++) {
      _staticSum += value[i];
    }
  }
  private int _sum;
  private static int _staticSum;
}

Tip: If possible, use local variables when accessing variables frequently.

Right

public class ULV {
  void oper1 (int[] values) {
    int sum = _sum;
    for (int i=0; i < value.length; i++) {
      sum += value[i];
    }
    _sum = sum;
  }
  void oper2 (int[] values) {
    int sum = _staticSum;
    for (int i=0; i < value.length; i++) {
      _staticSum += value[i];
    }
    _staticSum = sum;
  }
  private int _sum;
  private static int _staticSum;
}

REFERENCE

Peter Haggar: "Practical Java - Programming Language Guide". Addison Wesley, 2000, pp.122 - 125
