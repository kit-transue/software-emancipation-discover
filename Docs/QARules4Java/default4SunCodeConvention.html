EJB API Coding standard violations
Severity Abbreviation Explanation
High EJBCL The enterprise bean must not attempt to create a class loader, or use classloader object.
High EJBCCON The enterprise bean must not attempt to interact directly with a keyboard/display attached to the server system
High EJBBCNDA A Bean class cannot be declared as abstract.
High EJBECMBP ejbCreate () must be public, and can be neither static nor final
High EJBFDR The enterprise bean must not attempt to directly read or write a file descriptor.
High EJBIO The enterprise bean must not use the AWT functionality to attempt to output information to a display, or to input information from a keyboard.
High EJBIOFILES The enterprise bean must not use the java.io package to attempt to access files and directories in the file system.
High EJBJDBC The enterprise session bean should not work with persistent data.
High EJBMNDF A Bean class must not define the finalize() method.
High EJBPCMBP The ejbPostCreate () method must be public, and it cannot be static or final
High EJBREFL The enterprise bean must not attempt to use the Reflection API to access information that the security rules of the Java programming language make unavailable.
High EJBRTEB The finder method's type must be the primary key or a collection of primary keys.
High EJBRTSBV The SessionBean's ejbCreate() method's return type must be void
High EJBRTPCV The ejbPostCreate() method's return type must be void.
High EJBSOCKET The enterprise bean must not attempt to set the socket factory used by ServerSocket, Socket, or the stream handler factory used by URL.
High EJBSUBST The enterprise bean must not attempt to use the subclass and object substitution features of the Java Serialization Protocol.
High EJBTREST Avoid starting, stopping, or managing threads in any way.
Normal EJBAEBS Avoid access EJB entity bean directly from client code
Normal EJBAMSCO Avoid accessing or modifying security configuration objects
Normal Ejbauauf Always use accessors to update JavaBeans properties.
Normal Ejbbcdp A Bean class should be declared as public
Normal EJBIECM A Bean class should implement one or more ejbCreate () methods
Normal EJBIEPCM A EntityBean class should implement one or more ejbPostCreate () methods
Normal EJBLNLSBA Avoid loading native libraries in a Bean class
Normal EJBMFS Declare all static fields in the EJB component as final
Normal EJBTHIS Avoid passing the this reference as an argument
Normal EJBTHISRET Avoid returning the this reference as a result.
