Find Text dialog box

This tool is intended for searching strings in ‘Source’ page. It works only when this page is active. Use the Find Text dialog box to specify text you want to search for. To open the Find Text dialog box, either choose Edit | Text Find or press CTRL+F. To continue searching for previously specified text, either choose Edit | Find Again or press F3.



Text to find

Enter a search string; or, click the down arrow next to the input box to select from a list of previously entered search strings.

Options

Specifies attributes for the search strings.

Case sensitive

Differentiates uppercase from lowercase when performing a search.

Search only selected text

The search occurs only within text selected in the editor.

Search from start (end) of file

The search begins at either the start or the end of the file (depends on direction specified in the dialog).


If this and previous options are not checked, the search begins from the current position in the file.

Direction

Specifies in which direction you want to search.

Forward

Searches from the current position to the end of the file. Forward is the default Direction setting.

Backward

Searches from the current position to the beginning of the file.

Buttons

Find

Begins the search for the specified text. The search ends when the first instance of the search string is found (to continue searching use Find Again).

Mark All

Begins the search for the specified text from the start of file. The search ends when all instances of the search string are found. Each instances of the search string in editor are marked with green color. The only one option ‘Case sensitive’ influents on this action.

Cancel

Closes the dialog without any searching.
