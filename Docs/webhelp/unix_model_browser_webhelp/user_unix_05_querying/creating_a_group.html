Creating a Group

1   Select one or more items in one of the Browser or Minibrowser list regions.

2   Do one of the following:

•   Select Group > Create Group.

•   Click Group.

3   Enter a name for the new group.

4    Click OK.

This creates an empty group with the name RESULT_n, where n is a numeral that starts at 1 and increments with each group that you create. You can then add elements to the group.

You can name groups as you create them by entering a name in the Name field before you select Group > Create.
