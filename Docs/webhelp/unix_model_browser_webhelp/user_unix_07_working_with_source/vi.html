vi

You can choose to have vi as your standard integrated editor.

When using the DIScover text view in vi mode use the :e! command rather than :recover.

Use the Emacs C-x C-w command rather than the filename option to the :w command.

DIScover remaps built-in functions if they conflict with customized key bindings specified in the .exrc file. For example, if you map ~r to execute a macro DIScover remaps the ~ built-in function to ~~. A warning appears to notify you of this remapping upon startup of vi mode.
