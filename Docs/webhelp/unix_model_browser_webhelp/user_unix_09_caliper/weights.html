Weights

Each filter set and each query is assigned a weight. The weight represents an itemâ€™s relative importance among its peers. The more weight an item has, the greater its effect on the total score. Weights appear in the Weight box.

The default weight for every item is 10. You might choose to increase the weight of an item if you believe that its level of quality is of particular importance.
