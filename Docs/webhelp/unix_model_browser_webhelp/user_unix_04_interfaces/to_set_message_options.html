To set message options

1   Select File > Messages.

2   Toggle Use Message Logger on or off to choose whether or not to display messages.

3   Make one or more selections in the Message Filters section to choose which message types to display.

4   Make one or more selections in the Job Categories section to choose which tasks you want to see messages for.

5   Click Apply to save your settings.

6   Click OK.

•   New Browser launches a new Browser window.

•   Close closes the active window.

•   Quit exits DIScover.
