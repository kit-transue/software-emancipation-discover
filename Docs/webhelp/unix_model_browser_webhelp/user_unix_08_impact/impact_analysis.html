Impact Analysis

DIScover provides you with a unique opportunity to understand the impact proposed changes will have on your source code—before you make them. This book details the Impact Analysis feature, including impact options and instructions for proposing changes.

This book covers the following topics:

•   Why Use Impact Analysis?

•   Performing an Impact Analysis

•   Impact Analysis Options

•   The Impact Analysis Report

•   Attaching Impact Analysis Reports to Integrity Manager Issues
