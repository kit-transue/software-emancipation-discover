Rational ClearCase Specific Commands

•   Reserve, similar to Lock, creates a reserved copy of the file so that no one else can make changes while you have it checked out.

•   Unreserve removes the reservation from a checked out file.

•   List Checked Out displays all files which are currently checked out.

If your organization supports a custom CM system, your CM menu options may be different. For more information, see your administrator.

You can also check in files using the Submit button in the Submission Check window.
