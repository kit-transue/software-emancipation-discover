Inheritance View

The inheritance view displays class inheritance hierarchies. Using the shortcut menu, you can limit views of this type to the class level (including superclasses and subclasses), or you can include class members, as the following series of figures illustrate.


 

You can specify a different current class, displaying the inheritance relationships of superclasses and subclasses.


 

Class members are displayed with color coding to indicate public, private, and protected status. Using the right-click menu, you can also manipulate what the diagram displays. You can also set a number of filters to restrict the number and type of members that appear.
