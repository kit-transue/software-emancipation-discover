Change Propagation

TPM Change Propagation (CP) is a facility for automatically modifying source text, based on the structured information available in the Information Model, particularly in the Abstract Syntax Tree. You can use this tool to automate tedious and error-prone operations. Conversion scripts that are based on the program structure instead of text strings are much more reliable than automated conversions using text-driven tools like sed or perl that do not model the source syntax.
