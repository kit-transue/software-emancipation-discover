Samples of Access Programming

The following samples of Access programming are used with the DIScover Learn environment.

Note: When developing scripts, it is easiest if you work on a model that is not too big: it is helpful to be able to scroll back to see your last couple of queries and to correlate the query with the result set. As you progress in expertise, you will develop ways of interacting with large result sets, and, of course, when running scripts on your production model the question of what fits on the screen becomes secondary to the problem at hand.
