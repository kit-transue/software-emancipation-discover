About This Online Help

The following section details individual book content:

•   Understanding Access

Defines Access, offers a brief history of the tool, and outlines some of its key applications. The chapter concludes with samples of Access programming that use the DIScover learn environment.

•   Getting Started With Access

Explains how to use Access in Unix and Windows environments.

•   Access Fundamentals

Presents key Access concepts such as available data types, attributes and attribute expressions, sets and groups, and common procedures.

•   Tree Pattern Matching

Explains DIScover Tree Pattern Matching (TPM) technology and how it can be used to analyze vast regions of code for potential programming flaws. It includes a discussion of the Abstract Syntax Tree (AST), the data structure that serves as a foundation for all TPM technology. It presents TPM find procedures and conventions, as well as tips for writing effective TPM expressions.

•   Change Propagation

Explains how to use TPM technology to automate global changes to your source code. It includes a discussion of Access diff files, how to view, merge and apply them, and resolve conflicts that may occur during a change procedure.

•   Source Mode Queries

Presents the key concept of the Source Mode Tree (SMT), the mechanism DIScover uses to map between the source text and the elements in the information model. The chapter also includes a discussion of SMT navigation, attributes, and standard operations.

•   Access Commands

Presents a comprehensive list of available Access commands and attribute expressions.

•   Attribute Expression Grammar

Presents typographical conventions for working with Access and outlines TPM expression syntax and variables.

•   AST_node Attributes

Provides a comprehensive list of ast_node attribute codes as well as samples for C++, Java, and SQL programming languages.

•   Access Quick Reference

Provides a key to most commonly used Access commands, queries, and attributes.
