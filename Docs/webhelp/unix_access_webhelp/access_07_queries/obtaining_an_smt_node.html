Obtaining an SMT Node

The get_smt command converts a symbol or an instance to an SMT node that corresponds to that symbol’s definition or the span of source represented by the instance.

get_smt

get_smt symbol|instance

Returns the SMT node that spans symbol or instance.

Note: The data command—which converts between symbols, instances, AST nodes, and data nodes—is not used to convert to or from SMT nodes.
