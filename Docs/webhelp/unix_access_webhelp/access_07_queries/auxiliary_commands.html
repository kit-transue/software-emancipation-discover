Auxiliary Commands

The following commands may also be useful for SMT/CP:

cm_get

cm_get module_name

Makes module_name available for writing using the CM interface (checkout).

save

save module_name [module_name1...]

Writes the changes made during SMT/CP to disk. Note that module_name must be a string, not a DIScover handle or set. You should use the name attribute when saving a DIScover module: save [apply name $module].
