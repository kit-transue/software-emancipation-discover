Using Access in a Viewer

In DIScover, Access is available from an Emacs session in the Viewer. You need to open an Emacs session and invoke Access mode. Access mode uses Tcl mode as its foundation, incorporating Tcl functionality with the functionality of Access. As a result, you can use Access mode as you would use Tcl mode in any Emacs session.
