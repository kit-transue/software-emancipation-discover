TPM

The TPM menu provides shortcuts to tree pattern matching features, specifically, working with diff files. For more information about diffs, see Diff Initialization and Finalization and the following books in Change Propagation.

TPM menu options include:

•   Files with diffs returns files a TPM/CP operation has requested a change for.

•   View/Merge diffs opens the Merge user interface to view diffs.

•   Apply diffs makes changes to your source code according to the diffs present.

•   Remove diffs removes pending diff files.
