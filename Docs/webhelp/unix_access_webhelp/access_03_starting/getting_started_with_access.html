Getting Started With Access

This book provides instructions for launching Access on UNIX and Windows, including working within an interactive session and from the command line in both environments. It also provides information on customizing your UNIX user interface to add menu items and functionality.

This book covers:

•   Interactive Access Sessions on UNIX

•   Access from the UNIX Command Line

•   User Interface Customization on UNIX

•   Interactive Access Sessions on Windows

•   Access Menu
