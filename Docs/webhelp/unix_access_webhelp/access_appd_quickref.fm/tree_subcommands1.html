Tree Subcommands

Command

Action

tree filter attribute-expression scope

Return a set of nodes that match the specified attribute-expression.

tree find attribute-expression ast_node

Apply the specified attribute-expression to the specified AST node and its descendants. If the attribute-expression returns true, that node is added to the return set.

tree navigate attribute-expression scope

Apply the specified attribute-expression to each node in scope and return a node each time the attribute-expression succeeds.

tree print ast_nodes

Print a graphical representation of the specified AST node and its descendants.
