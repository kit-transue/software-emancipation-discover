Simple Examples

The following pattern uses iteration to locate plus expressions that operate directly on the integer constant 1900:

plus_expr -> repeat(,) "1900"

After locating a plus expression and moving to its first child, the cname of the child is compared to 1900. For more information, see Shortcuts. If it does not match, the same comparison is done for each sibling until a match is found or there are no more siblings left to compare.
