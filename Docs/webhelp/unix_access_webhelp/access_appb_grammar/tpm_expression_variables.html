TPM Expression Variables

<tpm term> Æ   

<string expression> |
<boolean expression> |
<unquoted string> |
<quoted string> |
<variable reference> |
<variable assignment>

<directional operator> Æ

-> |
<- |
, |
--

<repeat expression> Æ

r <subtree> <tree> |
repeat <subtree> <tree> |
<integer> <subtree> <tree> |
<integer> .. <subtree> |
<integer> ! <subtree>

<subtree> Æ

<directional operator> |
( <tree> )

<variable reference> Æ

$<identifier>

<identifier> Æ

<letter> { <letter or digit> }

<letter or digit> Æ

<letter> | <digit>

<letter> Æ

a | b | ... | z |
A | B | ... | Z

<digit>

0 | 1 | ... | 9

<variable assignment> Æ

<identifier> :

<string expression> Æ

s . <expression> |
<unquoted string> |
<access C-function attribute expression> |
<trinary operator>

<trinary operator> Æ

<expression> ? <expression> : <expression>

<unquoted string> Æ

<letter> { <letter or digit> }

<integer> Æ

<digit> | <digit> <integer>

<boolean expression> Æ

<expression> == <expression> |
<expression> != <expression> |
<expression> <= <expression> |
<expression> >= <expression> |
<expression> < <expression> |
<expression> > <expression> |
<expression> && <expression> |
<expression> || <expression> |
<expression> << <expression> |
<expression> >> <expression> |
! <expression> |
[ <access command> ] |
( <tree> ) |
find ( <tree> ) |
search ( <tree> ) |
find1 ( <tree> ) |
search1 ( <tree> ) |
{ <optional access command> } |
. |
nil

<expression> Æ

<integer expression> |
<boolean expression> |
<string expression> |
<quoted string> |
<integer>

<integer expression> Æ

+ <expression> |
- <expression> |
<expression> * <expression> |
<expression> / <expression> |
<expression> + <expression> |
<expression> - <expression> |
<expression> % <expression> |
<array reference>

<array reference> Æ

<string expression> [ <expression> ] |
<string expression> [ <expression> , <expression> ]

<quoted string> Æ

"Any characters"

<optional Access command> Æ   

{ <access command> }

<Access command> Æ

Any valid Access command.

<Access C-function attribute expression> Æ

Any Access C-function attribute expression, such as regexp or match.
