Working With Graphical Views

In addition to querying your source code, DIScover can assist you in better understanding it by displaying entities in graphical views, such as tree diagrams, entity relation diagrams, and flow charts.
DIScover’s graphical views make it easier to track function calls, relationships between entities, and more.

This book covers the following topics:

•   Graphical Views

•   Tree Diagram

•   Inheritance View

•   Outline View

•   Entity Relation Diagram

•   Data Chart

•   Flow Chart

•   Call Graph
