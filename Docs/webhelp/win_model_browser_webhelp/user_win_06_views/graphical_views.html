Graphical Views

When you choose an view, a View window appears, displaying the selected entities in a dynamic perspective of your application. Not all view formats are appropriate for all entity types. Appropriate views are also available through a shortcut menu when you select a given entity.

The model browser offers the following graphical views:

•   tree diagram

•   inheritance view

•   outline view

•   entity relation diagram

•   data chart

•   flow chart

•   call graph
