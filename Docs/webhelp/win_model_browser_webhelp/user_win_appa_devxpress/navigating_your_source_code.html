Navigating Your Source Code

You can navigate your source code by making one or more selections in the project hierarchy, then viewing the results in the runtime list. Locating files and specific source code entities allows you to perform queries or other actions on them.

When you start a Developer Xpress session, the project hierarchy displays all the directories directly beneath the directory designated as the root model directory (as defined by *dislite.ModelSrcRoot in your Application Setting file).
