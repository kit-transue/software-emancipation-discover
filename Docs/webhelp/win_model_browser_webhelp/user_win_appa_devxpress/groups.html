Groups

The group utility enables you to perform set operations on groups and save the results in a local directory. Clipboard is a scratch (temporary) group. To save the Clipboard contents to a persistent (permanent) group, select File > Save As and enter a group name.

To create new groups, do one of the following:

Select

To

File > New Group

Create an empty group or a group with the selected entities or files from the display panel.

File > Save As

Create a group with the selected entities or files from the display panel. If nothing is selected, the contents of the display panel are assigned to the group.
