Choosing a Configuration Management System

Out of the box, DIScover integrates with two configuration management systems, Source Integrity and Rational ClearCase. If you have a different CM system, your DIScover administrator can customize DIScover to enable integrated source control options.

Integration with a CM system allows you to check out, check in, and synchronize source files when you need to make modifications to your source files. Interaction with your selected CM system is available through a client CM menu, and its available options depend on your selected system. For more information, see Working With Configuration Management Systems.
