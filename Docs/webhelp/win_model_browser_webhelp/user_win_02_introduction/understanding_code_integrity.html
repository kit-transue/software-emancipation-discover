Understanding DIScover

DIScover is a unique set of tools that allows you to gain an unprecedented understanding of your source code so that you may effectively continue development and maintain quality standards as you complete your software projects. This book details DIScover features and functionality and introduces you to some key concepts in working with the Enterprise Edition of the product.

This book covers the following topics:

•   DIScover Overview

•   Core Concepts

•   DIScover Components

•   DIScover in the Enterprise Environment
