Overview of Dormant

Dormant analyzes a user-specified root entity within a selected project and determines which entities that root entity uses, directly or indirectly.

Based on your selections of project and root entity, DIScover identifies dormant entities within the project: functions, variables, macros, typedefs, structs, or classes that the root entity does not use.

DIScover creates a group containing the dormant entities, and stores it according to the Group Path preference. You can view resulting dormant analysis group members through the Group Manager.

Performing a dormant code analysis entails three tasks:

•   selecting a root entity (or entities)

•   performing the analysis

•   evaluating the results
