Dormant

Dormant is a source code analysis utility that identifies unused source code entities in your application. Dormant is a large-scale tool, designed to analyze entire applications, regardless of the number of lines of code.

It is important to note that Dormant performs a static analysis of your code; before you make changes based on such an analysis, you should perform tests to verify that the code that Dormant identifies as dormant is indeed dormant at run time.
