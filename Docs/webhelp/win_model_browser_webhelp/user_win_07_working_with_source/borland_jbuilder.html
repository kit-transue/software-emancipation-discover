Borland JBuilder

If you are using Borland JBuilder as your IDE, these functions can be accessed either by selecting options from the DIScover window, or by using icons from the add-in toolbar. When you are working with DIScover from within Borland JBuilder, DIScover messages appear in the Messages window in the bottom pane of your studio. If the Messages pane does not appear in your studio environment, select DIScover Worktray > Message Window from the menu bar.
