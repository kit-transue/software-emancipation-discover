Refining Your Source Analysis

You can select elements in the source window and bring up specific items of interest in the Browser pane. From the Browser, you can continue to perform analysis on the selected entity.

For example, if you opened a class file in a source window and notice a typedef reference that you want to examine further, you can open the typedef in the model browser and subsequently identify its related macros and so on.
