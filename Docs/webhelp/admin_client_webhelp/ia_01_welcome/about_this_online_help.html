About This Online Help

The following section details individual section content:

•   Understanding DIScover

Introduces the Code Integrity suite of tools, and provides the information a system administrator needs to know before planning a deployment. Describes a typical enterprise installation, and discusses administrative considerations and responsibilities.

•   Getting Started With an Information Model

Discusses the importance of the DIScover information model and includes tips to help you plan your model deployment. Introduces the central set of procedures used to create, manage, and maintain models and model servers.

•   Managing an Information Model

Explains how to perform incremental model builds, edit model configuration, delete models and model versions, resume a model build, analyze logs and debug models, and validate a model.

•   Submission Check

Describes how to setup and manage the submission check feature for Developer Xpress.
