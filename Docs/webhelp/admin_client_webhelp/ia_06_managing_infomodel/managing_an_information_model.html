Managing an Information Model

Once you build your first information model, it is your job as a
DIScover administrator to update versions of existing information models, build new models when necessary, analyze and debug models, start and stop model servers, and perform regular housekeeping of all information models.

This section covers the following topics:

•   Editing Model Configuration Information

•   Deleting Models and Model Versions

•   Analyzing Logs and Debugging Models

•   Validating Your Model

•   Stopping a Model Version

•   Checking and Fixing Model Servers
