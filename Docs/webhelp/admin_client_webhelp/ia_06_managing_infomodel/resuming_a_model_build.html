Resuming a Model Build

When a model build is interrupted, you have the option of resuming the model build process from where the interruption occurred. Resuming a model build is useful if the event that interrupted the build can be resolved without changing the model configuration. For example, the model build might run out of disk space, a temporary network problem may occur, or you might cancel the model build, preferring to start it at another time.
