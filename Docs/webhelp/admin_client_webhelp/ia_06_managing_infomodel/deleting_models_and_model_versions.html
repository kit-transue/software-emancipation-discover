Deleting Models and Model Versions

Similar to standard file housekeeping, it is necessary to remove old models and model versions that are no longer valid or useful to provide the most efficient model environment and to prevent the waste of disk space.

Important: Before you delete a model or a model version, we recommend instructing all DIScover users to disconnect from the model you want to delete. Failure to do so may result in errors if users are viewing the model in the model browser or Developer Xpress.
