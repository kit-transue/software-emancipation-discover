Interface Commands

There is an additional set of commands that serve to communicate with the Browser or a user. These commands are:

Interface Command

Description

get_selection

Places the entities selected in the Browser Elements or Results columns into the current set.

get_selection

set_result

Copies the contents of the specified set into the Browser’s Results column.

set_result set_name

msg

Generates a message relevant to the current operation in the Browser’s status bar.

msg options

emacs_command

Executes an Emacs command.

emacs_command command args

open_view

Opens the source of the specified entity.

open_view entity_name

prompt

Generates a prompt dialog box (with the information text contained in text_string) that accepts text input.

prompt text_string


These commands use the following syntax:

get selection
set_result [set1 set2 ...]
emacs_command (emacs-command)
msg -error "text"
open_view [set1 set2 ...]
prompt [ -title message_text] [-errmsg error_text]
