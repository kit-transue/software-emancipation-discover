Assumptions

Before using DIScover, we assume the following about your knowledge and experience:

•   You fully understand the hardware platforms and operating systems you are installing DIScover on, that is, Windows, Solaris, and HP-UX.

•   You understand the programming languages used in your source code, that is, ANSI C/C++, Visual C++, K&R C, Java, Oracle SQL, Informix SQL, and Sybase SQL.

•   You understand the compilers used to build your software projects, that is, Microsoft Visual C++, Sun cc, Sun WorkShop C++, GNU C/C++, HP cc, HP aC++, and Sun SDK Java.

•   If you are using an Integrated Development Environment (IDE) with DIScover, for example, Borland JBuilder, you understand how to use it.

•   If you are using a Configuration Management (CM) system with DIScover, for example, Source Integrity, you understand how to use it.
