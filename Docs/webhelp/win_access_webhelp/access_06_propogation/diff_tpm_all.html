diff_tpm_all

The following example demonstrates how you can search for a function call, pdstrcmp, and replace it with a new function call, new_pdstrcmp.

Note: Executing this command does not change your source code; it creates a diff file containing the delta between your original code and the changes that the command represents.

diff_tpm_all {call_expr -> c: "pdstrcmp” {diff_replace "new_pdstrcmp" $c}} /selected_files

You can execute a single command or a series. You may find it convenient to create a script containing a series of commands. A separate diff file is created with each execution of diff_tpm_all.
