Access-Specific Commands

Command

Action

apply attribute set

Apply specified attribute to each entity or instance in specified set, remove duplicate entities or instances.

data [symbol | instance | node | cell] entity

Convert specified entity from one Access data type to another.

filter set filter

Prevent display of member matching criteria defined in specified filter, for the specified set.

print set

Print entities contained in specified set.

printformat [-s [-a | -d] format-string attribute-list

Determines format of printed output:

-s -a: sort in ascending order

-s -d: sort in descending order.

size set

Return size of specified set.

sort sort-attribute

Return specified set according to specified sort attribute (alphabetical is default).
