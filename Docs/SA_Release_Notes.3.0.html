DIScover 7.3


DIScover 7.3 is the latest version of the DIScover product family. 


New features in DIScover 7.3 include:


C/C++/Java parser improvements

Realize a 30% improvement in build time on small models and up to 60% on larger models 






Impact analysis

Additional tools for understanding the impact of change to software 

Assists developers in reviewing the points of concern related to a given change 

Additional tools for understanding the impact of change to software 

Assists developers in reviewing the points of concern related to a given change 

New reporting capabilities 

Documentation explaining the different ways the impact of a change is tracked 




Change Phase I, GUI, wizards, etc.

Create Tree Pattern Matching expressions with new graphical assistant (representation of syntax tree) 

Smart listing of available attributes 

Ability to convert text selection to expression 

"Apply" feature for testing new expressions 

Save and restore expressions 

Assistance in node marking for transformation 



Licensing (use license key manager)

Choose between named-user licensing, concurrent (floating) license, or site license options when purchasing DIScover. Flexible licensing allows organizations to choose licensing model that best suits their needs 

Improved ease of deployment 

Minimal contact required to evaluate product 

Facilitate volume distribution 

Context sensitive help facility

Gain information on the task at hand without having to read the entire manual 


Model build Project editor

Improved ease of evaluation and use 

Opens up the product to all Java code 

Modify imported flags 

Small to midsize projects may now be configured by hand 

Basic SIS support (c-i, c-o, re-sync)

Provides the convenience of checking source in and out from DIScover 

ClearMake support

Improved integration into ClearCase environment for building models 


Bundling demo model in distribution

Provide users with a sample model to quickly try out the product without having to use their own code 

Provide users with firewall difficulty a way to easily evaluate the product 

Provide a base model for sales/training material 

Build Model in batch mode

New ability to run model builds in background 

Helps ensure the model will be always up to date

Hardware and Software Requirements


DIScover works on the following platforms:

Windows NT

Windows 2000

This version of DIScover has the following minimum system requirements:

Pentium 200 with 128 megs of memory

Microsoft Visual C++, version 5.x, 6.x, or 7.x

Editor support:

Microsoft Visual C++

Emacs

VIM

JBuilder 3.5, 4.0, 5.0
